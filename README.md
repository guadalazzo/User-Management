# User managment

Uploaded in: user-managment-config.netlify.app

User config managment for teams, with role handling

## Start the project

Open the terminal in the repository location

Install packages:

```bash
 yarn
```

Run react server

```bash
yarn dev
```

open: http://localhost:5174/

Unit Testing - Jest

```bash
yarn test
```
